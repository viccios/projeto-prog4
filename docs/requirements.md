# Requisitos

## Funcionais
- O sistema deve permitir o cadastro de até 3 pets por usuário.
- O sistema deve permitir que o usuário se inscreva em cursos.
- O sistema deve possuir sistema de pesquisa de aulas e cursos.
- O sistema deve conter sistema de login e registro de usuários.
- O sistema deve recomendar aulas e cursos de acordo com os interesses do usuário.
## Não Funcionais
- O sistema deve ser compatível com computador e celular.
- O sistema deve possuir interface intuitiva.
- O sistema deve conter uma diversidade de cursos e aulas.
